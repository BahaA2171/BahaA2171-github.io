                  Projects

                  Smart Recruiter:

                  Objective:The Smart Recruiter project is an AI-powered virtual interviewer designed to assess candidates’ stress levels and emotional states during interviews by analyzing voice, facial expressions, and body language in real-time.

                  1. Interaction with Large Language Models (LLMs):

                  Description: The system integrates an LLM (using models like Llama3 and Phi) to simulate natural conversations. The virtual interviewer engages with candidates by asking tailored questions and providing real-time feedback.

                  Technologies Used:

                  • LLMs: Pre-trained large language models for natural language understanding.
                  • LangChain: Orchestrates interactions between the LLM and the candidate's responses.

                  2. Avatar Creation & Animation:

                  Description: a virtual avatar is used to simulate a human interviewer. The system dynamically changes the avatar’s lips to match the spoken words in real-time using OpenCV, ensuring accurate lip synchronization with the Text-to-Speech (TTS) output. This creates a realistic and engaging interaction.

                  Technologies Used:

                  • OpenCV: Used to modify the avatar's lips based on the spoken words.
                  • Text-to-Speech (TTS): Provides the speech output that the avatar mimics in real-time.

                  3. Stress Detection Models:

                  Description: Deep learning models assess stress based on eye metrics, head rotation, lip movement, body language, and voice analysis.

                  Body Language Model :

                  Technologies Used:

                  • MediaPipe: For body and facial tracking.
                  • Librosa: For audio feature extraction (MFCC, RMS, ZCR, Mel-spectrogram).
                  • TensorFlow & Scikit-learn: For DL and ML models.

                  4. API & UI Development:

                  Description:The system features a FastAPI-based backend that handles real-time stress and emotion detection, serving model outputs to the frontend. Additionally, a PySimpleGUI-based interface was developed to facilitate data collection, model training, and testing, allowing recruiters to interact with the AI system, view candidate reports, and visualize stress analysis results.

                  Key Features:

                  • Real-Time Analysis: Processes video and audio input to provide immediate feedback on candidate stress levels and emotional states during the interview.
                  • Data Collection: The interface allows users to collect and annotate real-time video, audio, and gesture data. Users can label data with stress/emotion tags for use in training and testing models.
                  • Model Training & Testing: Provides an interface to upload training datasets, run training sessions, and test models, offering insights into the accuracy and performance of the stress detection models.
                  • User-Friendly Interface: A graphical interface developed with PySimpleGUI allows recruiters and developers to upload video interviews, review reports, and train/test models seamlessly.

                  Technologies Used:

                  • FastAPI: For backend API development.
                  • PySimpleGUI: For building a user-friendly interface for data collection, training, and testing.
                  • TensorFlow & Scikit-learn: For building, training, and testing deep learning models used in stress detection.
